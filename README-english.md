# template-discord
a template for me when I want to create a bot discord

# Before starting the bot

## add the config file

Create a config.json file:
Enter your bot's token, your bot's id, the server id, and your own Discord account id (for adminId) in this format:

```json
{
  "token": "your Token",
  "clientId": "your ClientId",
  "guildId": "your GuildId",
  "adminId": "your AdminId"
}
```

### attributes 

- **token**: your bot's private key
- **clientId**: the bot's user id <br/> _(accessible by right-clicking and pressing "Copy User ID" on the bot's profile page)_
- **guildId**: your Server id <br/>_(accessible by right-clicking and pressing "copy Server ID" on the Server page)_
- **adminId**: your chat room ID <br/>_(accessible by right-clicking and selecting "Copy User ID" on your profile page)_
